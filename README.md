# Customer_Segmentation-02

##Tools used
- Excel
- Power BI
- Machine learning (Orange)
- Python

##Project overview
- This project analyzes customer shopping behavior using the Shopping Trends dataset.
- K-Means clustering is used to segment customers based on purchasing patterns to support targeted marketing strategies.

##What I learned
- How to clean and prepare data for analysis  
- How to perform customer segmentation using K-Means clustering  
- How to analyze customer purchasing behavior  
- How to visualize insights using Power BI
